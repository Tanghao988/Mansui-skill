# 02 — 角色深度分析 (Character Analysis)

## 社区评价

### 核心评价：知世故而不世故
经历世间最残酷的苦难——家人饿死、吃亲人肉、流浪三年、手刃仇敌——却没有黑化，依然保持善良本性。

### "不似羊的猫"
游戏官方定位。看起来温顺无害（像羊/小羊），实则机灵敏捷有利爪（像猫）。
- 猫的特质：聪慧、敏捷、狡猾、圆滑
- "敏慧"与"满穗"近音

### "坏女人"六特质（机核网分析）
1. **出众特质**：立绘精美，身材娇小面庞精致，与其他角色拉开显著差距
2. **主动拉近距离**：主动叫"良爷"、教影子戏、暧昧挑逗——"坏女人"而非"傲娇"
3. **神秘感**：谎话连篇，真假参半难以琢磨
4. **若即若离**：看似亲近又保持距离，让人捉摸不透
5. **让人产生保护欲**：乱世孤儿身份激发怜爱
6. **最终证明善良**：真结局中选择不杀良、甚至投湖自尽
